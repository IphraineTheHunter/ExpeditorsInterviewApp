The test suite can be run by executing `./gradlew test` from the root directory of this project

Assumptions
**Addresses are the unique combination of city, state, and street address
**all strings should be considered case insensitive
**streets, roads, avenues, etc. will all be abbreviated
**extra commas and periods in addresses are optional
**When reporting the number of residents, the full total should be reported, not just those of age
